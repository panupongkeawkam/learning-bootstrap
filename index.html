Overview

Bootstrap version 5.x.x คือ CSS Framework ที่ช่วยในการออกแบบและปรับแต่งไซต์ได้อย่างรวดเร็วและสามารถจัดการกับ Responsive Website ได้อย่างง่ายดาย โดยลดขั้นตอนการเขียนโค้ดในส่วน CSS ลงอย่างมากพร้อมทั้งมีการปรับแต่ง style หรือลูกเล่นการใช้ script มาให้พร้อมใช้งานได้ทันทีผ่านการเรียก class หรือ attribute

ซึ่ง Bootstrap เป็นชุดเครื่องมือโอเพนซอร์สส่วนหน้ายอดนิยมของโลก ซึ่งประกอบด้วยตัวแปร Sass และปลั๊กอิน JavaScript

ช่วยได้มากแค่ไหน?

หลักๆ นั้น Bootstrap จะช่วยลดการเขียนในส่วนของ style ลง โดยการเรียกใช้ style จะเรียกผ่าน class ต่างๆ ที่มีมาให้ ร่วมไปถึงการที่ตัว Bootstrap class บาง class มีการปรับแต่งตัว style เพิ่มเติมที่ทำให้ใช้งานในเหมาะสมยิ่งขึ้นและนอกจากนั้นยังมีการที่สามารถเรียกใช้งานตัวของลูกเล่น JavaScript ผ่าน attribute ของตัว tags เองได้เลย

และในส่วนของตัวอย่างการใช้งานคล่าวๆ ของการปรับแต่ง style โดย Bootstrap โดยเพียงเรียกผ่าน class ดังด้านล่าง

Containers & Responsive

Container เป็นส่วนประกอบพื้นฐานของ Bootstrap ที่บรรจุตัว element ลูกไว้ เป็นการ set ขอบเข็ตของตัวเว็บไซ์ และสามารถจัดการกับ Responsive และ Viewport ของเว็บไซต์ตามที่กำหนด

ทำงานอย่างไร?

Container หรือชื่อ class คือ .container-* เป็นการกำหนดขอบเขตของเว็บไซต์โดยจะกำหนดให้กับ element tag ที่ใหญ่ที่สุดเว็บไซต์ (รองจาก <body>) โดย property ใน .container-* จะเป็นกำหนด width: ตาม class container แต่ละแบบและกำหนดเป็น height: auto; โดยจะอธิบาย container แต่ละประเภทดังนี้

  • .container จะเป็นกำหนด width: ตามขนาดหน้าจอต่างๆ แบบอัตโนมัติ
  • .container-fluid จะกำหนดให้ทุกขนาดหน้าจอเป็น width: 100%; ทั้งหมด
  • .container-{ขนาดหน้าจอ} จะกำหนดให้ width: 100%; ก็ต่อเมื่อถึงขนาดจอที่เลือกไว้

ในส่วนของ .container ในแบบต่างๆ จะมีชื่อ class และการ breakpoint อิงตามตารางด้านล่างดังนี้

Grids & Layout

การจัดวาง element ของ Bootstrap จะเป็นกำหนดเป็น row และ column โดยในส่วนของ column จะสามารถกำหนดขนาดความกว้างได้ และในการใช้งานตัว column จะเป็น element ลูกของ row เสมอ และในส่วนของความสูงของ row หรือ column จะขึ้นอยู่กับความสูง (height:) ของเนื้อหาหรือ element ภายในอีกที

Row

Row จะมีชื่อ class ใน Bootstrap ว่า .row จะมีการกำหนด property เป็น width: 100%; ตาม element แม่ โดยจะมีจัดวาง element เป็นแนวแถว (row) และเมื่อ element ภายในเกินขอบเขตของตัวมัน มันจะขึ้นเป็นแถวใหม่ให้

Column

Column จะมีชื่อ class ใน Bootstrap ว่า .col โดย .col จะสามารถกำหนดความกว้างเองได้หรือว่าจะหารความกว้างตามจำนวน .col ใน .row ก็ได้

นอกจากนี้ยังสามารถกำหนดความกว้างที่ต้องการตามขนาดของหน้าจอตอนแสดงผลได้ และสามารถกำหนด column แบบต่างๆ ตามชื่อ class ได้ดังนี้

  • .col จะหารความกว้างตามจำนวนของ .col ภายใน .row
  • .col-{ความกว้างเป็นสัดส่วน (0 - 12)} จะกำหนดความกว้างเป็น 12 ส่วนตามเลขที่กำหนด ของ .row (จะขึ้นแถวใหม่หากเกิน 12)
  • .col-{ขนาดหน้าจอ}-{ความกว้างเป็นสัดส่วน (0 - 12)} column จะถูกกำหนดเป็นความกว้างที่กำหนดก็ต่อเมื่อถึง breakpoint ของขนาดหน้าจอนั้นๆ

Utilities

ในส่วนของ Utilities หรือการจัดวางระยะห่างของเนื้อหาภายใน (padding:), การจัดระยะห่างของตัว element (margin:) Bootstrap ก็จะทำให้ง่ายขึ้นด้วยเช่นกัน โดยจะลดการเขียนในส่วนของ style ลง ซึ่งจะเขียนในรูปแบบของ class แทน

Margin & Padding

ในการกำหนด margin: ของ Bootstrap จะใช้ ตัวย่อ 'm' และเช่นเดียวกับ padding: ซึ่งจะใช้ตัวย่อ 'p' และตามด้วยค่าที่ต้องการ โดยจะสามารถกำหนดทิศทางของ margin: และ padding: ได้เช่นกัน และนอกจากนั้นยังสามารถกำหนด margin: และ padding: ตามการแสดงผลหน้าจอได้เช่นเดียวกัน

โดยจะเขียนเป็น class ตามการกำหนดต่างๆ ได้ดังนี้

  • .m-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element รอบทิศทางเป็นค่าที่ต้องการ (e.g. m-3, m-5)
  • .p-{ค่า 0-5} กำหนดการจัดวางระยะห่างของเนื้อหาภายในรอบทิศทางเป็นค่าที่ต้องการ (e.g. p-2, p-0)
  • .{m, p}-{ขนาดการแสดงผลหน้าจอ}-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element หรือระยะห่างของเนื้อหาภายในรอบทิศทางตามค่าที่กำหนด เมื่อถึงขนาดการแสดงผลหน้าจอที่ต้องการ (e.g. m-xxl-3, p-sm-4)
  • .{m, p}t-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element หรือระยะห่างของเนื้อหาภายในส่วนบนเป็นค่าที่ต้องการ (e.g. mt-3)
  • .{m, p}e-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element หรือระยะห่างของเนื้อหาภายในส่วนขวาเป็นค่าที่ต้องการ (e.g. pe-5)
  • .{m, p}b-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element หรือระยะห่างของเนื้อหาภายในส่วนล่างเป็นค่าที่ต้องการ (e.g. pb-3)
  • .{m, p}s-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element หรือระยะห่างของเนื้อหาภายในส่วนซ้ายเป็นค่าที่ต้องการ (e.g. ms-2)
  • .{m, p}{ทิศทาง}-{ขนาดการแสดงผลหน้าจอ}-{ค่า 0-5} กำหนดการจัดระยะห่างของตัว element หรือระยะห่างของเนื้อหาภายในตามทิศทางที่กำหนดและตามค่าที่กำหนด เมื่อถึงขนาดการแสดงผลหน้าจอที่ต้องการ (e.g. mt-xl-5,
    pe-md-2)

โดยหน่วยของค่า 0-5 จะเป็นดั่งตารางด้านล่าง

Background & Colors

ใน Bootstrap ยังมีการกำหนดสีพื้นหลัง (background-color:), สีตัวอักษร (color:) มาให้ผ่านการเรียกใช้ class โดยสีที่กำหนดมาจะเป็นสีที่ค่อนข้างพื้นฐาน โดยอาจจะไม่ได้มีความยึดหยุ่นในการใช้แต่ก็เป็นตัวเลือกในการแยกสีใน element ได้อย่างรวดเร็ว

โดยค่าของสีต่างๆ ของ Bootstrap มีดังตารางด้านล่างนี้

สีพื้นหลัง

ในส่วนของสีพื้นหลัง (background-color:) จะมีการใช้ลูกเล่นการไล่สี (gradient) เพิ่มเข้ามาโดยไม่จำเป็นต้องตั้งค่าการไล่สีเอง โดยในการเขียนชื่อ class จะเป็นดังด้านล่าง

  • .bg-{ชื่อของสี} จะกำหนด background-color: เป็นสีที่ต้องการ
  • .bg-gradient


สีตัวอักษร

สีตัวอักษรจะสีค่าสีที่เหมือนกันกับ .bg-* และยังสามารถกำหนด opacity: ของตัวอักษรได้ แต่ได้เพียงแค่ 50%

  • .text-{ชื่อของสี} จะกำหนด color: เป็นสีที่ต้องการ
  • .text-{ชื่อของสี}-50 จะกำหนด color: เป็นสีที่ต้องการและให้ความโปร่งแสงของข้อความเป็น 50%

Border & Shadow

กรอบ (Border)

ในการกำหนดกรอบ (border:) ก็สามารถกำหนดได้ผ่านการเรียก class เช่นเดียวกัน ไม่ว่าจะเป็นทิศทางกรอบ ความหนา ขอบมน หรือค่าสีของกรอบ ซึ่งในค่าสีของกรอบก็อิงตาม ตารางสี เดียวกับการกำหนดสีพื้นหลังหรือสีข้อความ

โดยจะเขียนชื่อ class เพื่อกำหนด properties ต่างๆ ได้ดังนี้

  • .border กำหนดกรอบของ element รอบทิศทาง
  • .border-{ค่า 0-5} กำหนดความหนากรอบของ element เป็นค่าที่กำหนด
  • .border-top กำหนดกรอบของ element ด้านบน
  • .border-end กำหนดกรอบของ element ด้านขวา
  • .border-bottom กำหนดกรอบของ element ด้านล่าง
  • .border-start กำหนดกรอบของ element ด้านซ้าย
  • .border-{ชื่อของสี} กำหนดสีกรอบของ element ตามชื่อสีดังตารางสี
  • .rounded-{ค่า 0-3 หรือ ลักษณะความมน} กำหนดความมนของกรอบ element (border-radius:) เป็นค่าที่กำหนด

เงา (Shadow)

ในส่วนของเงาใน Bootstrap จะไม่ได้มีการกำหนดที่ละเอียดสักเท่าไหร่ โดยจะกำหนดแค่ขนาดของเงาเท่านั้น

โดยจะเขียนชื่อ class เพื่อกำหนดขนาดเงาได้ดังนี้

  • .shadow กำหนดเงาของ element เป็นขนาดกลาง
  • .shadow-sm กำหนดเงาของ element เป็นขนาดเล็ก
  • .shadow-lg กำหนดเงาของ element เป็นขนาดใหญ่
  • .shadow-none กำหนดให้ไม่มีเงาใน element

Code Example:

<div class="shadow-none">.shadow-none</div>
<div class="shadow-sm">.shadow-sm</div>
<div class="shadow">.shadow</div>
<div class="shadow-lg">.shadow-lg</div>

Table

การใช้ class จาก Bootstrap ในการแสดงผลตาราง ถือว่าค่อนข้างที่จะลดความซับซ้อนของโค้ดไปเยอะมาก เนื่องจากการสร้างตารางของภาษา html ค่อนข้างจะกำหนด style ที่ยุ่งยาก ซึ่ง Bootstrap ก็มี class ที่ใช้เพียงแค่ไม่กี่ class ก็สามารถที่จะทำให้ตารางออกมาดูสวยและมี Responsive อีกด้วย

ซึ่งชื่อ class ต่างๆ และผลลัพธ์ที่ได้จะเป็นดังนี้

  • .table กำหนดโครงสร้างตารางที่เหมาะสมขึ้นมาโดยจะกำหนด property มาให้อัตโนมัติ (จำเป็นต้องใช้กับ tag <table>)
  • .table-bordered กำหนดให้ตารางมีเส้นขั้นในทุก cell
  • .table-borderless กำหนดให้ตารางไม่มีเส้นขั้นในทุก cell
  • .table-striped กำหนดให้สีของแต่ละ row มีความโปร่งแสงของพื้นหลังที่สลับกันไป
  • .table-{ชื่อของสี} กำหนดสีของตารางโดยชื่อสีอิงตามตารางสี

<table class="table">
  <thead>
    <tr>

      <th>Title3</th>
    </tr>
  </thead>
  <tbody>

      <td>Cell 00</td>
      <td>Cell 01</td>
      <td>Cell 02</td>
    </tr>
    <tr>
      <td>Cell 10</td>
      <td>Cell 11</td>
      <td>Cell 12</td>
    </tr>
  </tbody>
</table>

Navs & Tabs

สำหรับแทบบาร์ใช้สำหรับการนำทางไปยัง page ต่างๆ หรือ section ต่างๆ ของหน้าเว็บนั้น ย่อมจะมีในแทบจะทุกๆ เว็บ และ Bootstrap ก็ยังมีเครื่องมือในการจัดการอีกเช่นเคย โดยในการจัดวาง layout ของ navigator bar ค่อนข้างจะแตกต่างจากการจัดวาง element ทั่วๆ ไปพอสมควร

และสำหรับการกำหนด property ตามชื่อ class ต่างๆ จะมีดังนี้

  • .nav เป็นการกำหนดเพื่อเป็น element tab bar หลัก โดยจะต้องใช้คู่กับ tag <ul>
  • .nav-item เป็นการกำหนดตัว element เป็นปุ่มครอบในการนำทาง โดยจะต้องใช้คู่กับ tag <li>
  • .nav-link เป็นการกำหนดให้ตัวข้อความให้สามารถนำทางได้ โดยจะต้องใช้คู่กับ tag <a>
  • .nav-fill เป็นการกำหนดให้ตัวปุ่มภายใน tab bar มีการกระจายความกว้างของปุ่มให้เท่ากัน
  • .nav-pill เป็นกำหนดให้ตัวปุ่มนำทางมีการขึ้นรูปร่างปุ่มเมื่อสถานะถูกใช้งาน
  • .active เป็นกำหนดให้ปุ่มนั้นมีสถานะการถูกใช้งาน
  • .disabled เป็นกำหนดให้ปุ่มนั้นมีสถานะการถูกปิดใช้งาน

Code:

<ul class="nav">
  <li class="nav-item">
    <a class="nav-link active" aria-current="page" href="#">Item1</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Item2</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Item3</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Item4</a>
  </li>
</ul>


Result:

  • Item1
  • Item2
  • Item3
  • Item4
